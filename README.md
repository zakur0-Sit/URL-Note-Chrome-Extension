# URL-Note-Chrome-Extension

URL Note is a simple Chrome extension for saving and managing links directly from your browser. You can save links manually or from the active tab, all data is stored locally for quick access.

## Features

- **Manually add links**: Input a link in the text box and click **SAVE INPUT** to save it.
- **Save active tab's link**: Click **SAVE TAB** to save the URL of the active tab.
- **Delete all saved links**: Double-click **DELETE ALL** to remove all saved links.
- **View saved links**: All saved links will be displayed in a list, accessible by clicking each link.

## Installation

1. Clone this repository or download the files.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the folder containing the extension files.
5. The extension will be added to your browser and should appear in your toolbar.

## Usage

- Enter a link in the input field and click **SAVE INPUT** to save it.
- Click **SAVE TAB** to save the active tab's URL.
- Double-click **DELETE ALL** to remove all saved links.
- All saved links will be listed and can be opened in a new tab by clicking on them.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage (for saving data)
